# SRE Production Platform

Este projeto demonstra a criação de uma plataforma Kubernetes em produção utilizando AWS, Terraform, EKS, GitOps e ferramentas de observabilidade.

## Tecnologias

- AWS
- Terraform
- Kubernetes
- Docker
- Helm
- ArgoCD
- GitHub Actions
- Prometheus
- Grafana
- Loki
- Fluent Bit

## Objetivos

- Provisionar infraestrutura com Terraform
- Criar um cluster EKS
- Automatizar Deploy
- Implementar GitOps
- Configurar Observabilidade
- Configurar Auto Scaling
- Simular incidentes
